# Campus Fintech Wallet

A digital payment and financial management system designed specifically for college and university campuses.

## Features

- Student digital wallet with QR payments
- Vendor payment acceptance system
- Admin dashboard for management
- Transaction history and reporting
- Cashless campus transactions

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Authentication**: JWT
- **Database**: In-memory storage (for demo)

## Quick Start

```bash
# Install all dependencies
npm run install-all

# Start both frontend and backend
npm run dev
```

The frontend will be available at `http://localhost:3000`
The backend API will be available at `http://localhost:5000`
